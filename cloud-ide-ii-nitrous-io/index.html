Cloud IDE II - Nitrous.IO

Nitrous.IO is the last of the three IDEs that I found. I fact, I just registered myself the day before I started writing this review. I was introduced to Nitrous.IO by a mentor quite a while ago, but I couldn’t manage to get an invitation for the private beta. So, I waited until public beta to try it out.

Pricing

Each virtual machine in Nitrous.IO is called a box. Users can purchase or upgrade boxes using a currency called N2O. At the time of writing, each user is given 150 N2O for free, and more N2O can be earned by various actions like linking account to GitHub, inviting friends and so on. The initial 150 N2O is enough to purchase a bare minimum box with 384 MB memory and 750 MB storage. Additional memory costs 40 N2o per 128 MB block and additional storage costs 10 N2O per 250 MB block. Although the price to purchase N2O has yet to be announced, from the pricing page, it is easy to deduce that 10 N2O costs $0.35. I am confused right here. According to the Nitrous.IO introduction video, N2Os last forever and will be returned when a box is destroyed. (For example, if a box that costs 150 N2O is destroyed, user will get back that 150 N2O.) But from the pricing page, the tentative price is given as dollar per month. So, in the future, if users choose to purchase N2O, will these N2O last forever as a one-time purchase?

Virtual Machine

Each box can have a specific template running Ubuntu 12.04 LTS. At the time of writing, there are four templates available - Ruby on Rails, Node.js, Django and Go. I tried with the Go template, not understanding what template actually means. Well, each template will have all the necessary tools required to develop for that framework. For example, in the Go template, the Google App Engine SDK is installed by default. But that doesn’t mean that other languages cannot be used - I found GCC, Ruby, Python, Python 3, Java, and Go (of course). However, to get Ruby on Rails working, you will need to do your own work - install the Rails gem. At this point, I realize we do not have root access, but almost everything is available - Vim, Git, Subversion, Mercurial, SSH and other. As for SSH, both SSH from and to other machines is supported. For the sake of curiosity, I tried to install RVM. I manage to do so, but couldn’t install other versions of Ruby.

Overall experience

Personally, I enjoy using Nitrous.IO, besides the lack of root access is making me itchy. The text editor is a typical plain text editor with syntax highlighting for more than a dozen languages including C, C++, CSS, HTML, Go, Ruby, SQL, etc. (Even includes some languages that Nitrous.IO doesn’t give compiler/interpreter for like PHP.) One thing that surprises (and pleases) me very very much is the performance of the shell. As we will see with the other two IDEs, their shell is slow. I can easily type faster than the shell. Using vim on such environment is a pain. However, on Nitrous.IO, the shell is almost as fast as the Terminal of my Mac or some high-end VPS.
