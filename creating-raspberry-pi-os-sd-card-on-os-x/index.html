Creating Raspberry Pi OS SD Card on OS X

I’ve been using Mac for about four months now, and I am ashamed to say that I couldn’t successfully create a Raspberry Pi SD card on this machine, not until today. The usual problem that I got was:

dd: /dev/rdisk1: Permission denied

Finally I found the solution today.

  • Open up Disk Utility. If the SD card that you intend to use has a few partitions, click on your SD card, click partition. On the drop-down, select 1 Partition and click Apply. Disk Utility will now wipe your SD card and modify the partition so that there is only 1 on the card.
  • Go to https://github.com/RayViljoen/Raspberry-PI-SD-Installer-OS-X. Either do a git clone or download and extract the zip of the whole repository.
  • Download an OS for Raspberry Pi here.
  • Make sure the checksum for the downloaded archive matches with the one displayed on the download page.
  • Navigate to the repository downloaded in step 2.
  • sudo ./install ~/path/to/downloaded/os
  • Select the correct device.
  • Wait.
  • You’re done.

Please take note that step 1 is important. Step 1 is the reason that I got permission denied for the dd command.
