Chromebook for serious business

I was loaned a Samsung Chromebook when I went to the Malaysian APIO training camp in early May. At first sight, it is thin, looks a lot like the MacBook Pro without the Samsung and Chrome logo on the lid. The operating system of this machine is Chrome OS, which essentially is just Google Chrome. Anyway, it is Linux based, hence, in theory, it is possible to run any Linux distro as long as the hardware is capable of handling it. As a matter of fact, there are already two versions (AFAIK) of Ubuntu available - Crouton and ChrUbuntu. I have not tried either, so I have nothing to talk about this.

My job with the Chromebook was to evaluate if it is suitable for students to use it as a coding machine. Well, my first impression was that Chromebook requires an internet connection to make it useful - without an internet connection, only a very limited selection of apps is able to function. So, unless we can find a good web-based IDE, this is definitely out of talk. Frankly, I have no high hope on this. Anyway, I proceed to do a lot of Googling to find a good web-based IDE. Here are my requirements

  • C++ and Python support
  • A decent code editor that has syntax highlighting and the usual stuff
  • Preferably git and shell
  • Offline support (at least for code editing)

It turns out that most of the web-based IDE does not support non-web-based languages like C++ and Python - most of them are for HTML, CSS, JS and the likes. However, I did found a web-based IDE that scored 75% (3 out of 4) criteria (no offline support) - Cloud9 IDE. Not only it has git support, it can directly clone a repo from GitHub or BitBucket, and the shell is almost a 100% Unix shell - it supports chmod, wget, time, vim, etc. However, the shell is a bit laggy, making compilation and debugging somewhat unpleasant.

I am still looking for other web-based IDE that might potentially score 100%. I know there are some offline code editor for Chromebook, but it can be quite inconvenient as there is no direct integration between such editors and Cloud9 IDE or git.

In conclusion, I would say that Chromebook is a so-so coding machine for students especially when working with languages like C++ and Python (other web-related languages might be better). I was asked, “Will you switch from your existing machine to Chromebook when you attend a similar training camp?” (FYI, what we do at the training camp is basically some algorithmic programming in C++.) The answer? How I am supposed to give up my MacBook Pro and switch to this machine? I would prefer Raspberry Pi over Chromebook for similar purpose. (Will talk about this later on.)
