Ruby on Rails

I’ve been focusing on C++ programming for quite a while. Indeed, C++ is my main programming language, but I am looking to pick up a second language that I can work as comfortably as I do with C++. The first two candidates are Python and Java. I figured that Java is somehow too similar to C++, that means no fun spending time learning it, at least not yet. I’ve finished the tutorials for Python on codeacademy, but I don’t think I can write any program that I can write on C++ with less than 5 Googles. So, I decided to try Ruby. One of the reasons for choosing Ruby is that it is quite similar to Python, and my mentor on web dev actually encourage me to learn Ruby and use Ruby on Rails to develop web apps. After more discussion with her, I decided to start with Ruby on Rails.

More updates on Rails to come.
