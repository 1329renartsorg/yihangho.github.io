Reinstall OS X Mountain Lion (and probably other versions too)

  1. Make backup copies of all important files.
  2. Shutdown completely.
  3. Start the computer while holding the option key until a screen with 2 (or possibly more) hard disks shows. Select Recovery HD.
  4. Select Reinstall OS X. Follow the instructions until you arrive at the screen where you are asked to select the partition in which OS X should be installed to. You should choose Macintosh HD. If you are able to select Macintosh HD, continue and finish the installation process.
  5. If you are unable to select Macintosh HD in the previous step and are told that “This disk is locked”, then quit the reinstaller and choose Disk Utility on the initial menu. Select Macintosh HD and format it. Take note to format only the Macintosh HD partition, but not the whole drive. Repeat step 4.
