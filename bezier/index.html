Bézier

A few days back, I published a Ruby gem, Bezier. Bezier is a library that can draw Bézier curves and produce PNG (thanks for ChunkyPNG).

Using Bezier is super simple. First, install it:

1gem install bezier

then do some magic with your Ruby script:

 1require 'bezier'
 2
 3# Create some control points
 4control_points = [[20, 20], [20, 100], [200, 100], [200, 20]].map do |coord|
 5  Bezier::Point.new *coord
 6end
 7
 8# Create a Bézier curve using those control points
 9bezier = Bezier::Bezier.new *control_points
10
11# Produce the PNG at Bézier.png
12Bezier::Painter.new { |p| p.points = bezier.run }.save

Do try it out!
