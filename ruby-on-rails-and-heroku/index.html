Ruby on Rails and Heroku

As mentioned earlier, I am now learning Ruby on Rails with this super awesome tutorial by Michael Hartl. The author suggests that whatever code should be pushed to production environment as soon as possible, and the production environment that he suggested was Heroku. I did not do so until the mid of Chapter 6, and I face a problem immediately - all the static assets (images, JS, CSS) are not served. After much Googling and fiddling, the solutions was very simple:

Edit config/environments/production.rb: Set config.serve_static_assets to true.

Update

Apparently the author mentioned this in Chapter 2.
